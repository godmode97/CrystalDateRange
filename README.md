## Development

* Add dependencies on `shard.yml`

```yml
  dependencies:
    spec2:
      github: waterlink/spec2.cr
      version: ~> 0.9
    cronus:
      github: aisrael/cronus
      branch: develop
```

* run `shards install` or `shards update` (when updating dependencies)
